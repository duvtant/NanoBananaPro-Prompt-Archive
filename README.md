## ğŸ¤ Contribution Guidelines

This archive is powered by the community! We welcome your best creations.

### How to Contribute:

1.  **Fork** this repository.
2.  In the `prompts-and-images/` folder, create a new subfolder if your work fits a new category, or use an existing one.
3.  Add your generated image file (`.png` or `.jpg`).
4.  Add a corresponding text file (`.txt`) containing **only the prompt(s)** used.
5.  Create a **Pull Request (PR)** with a descriptive title (e.g., "Add 3 new landscape prompts to High-Res-4K").

**Note:** Only submissions generated specifically by the **Nano Banana 2 Pro / Gemini 3 Pro Image** model will be accepted.
